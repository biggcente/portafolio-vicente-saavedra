#  Portafolio Web Profesional

**Vicente Saavedra — Desarrollador Web Full Stack**

---

##  Descripción del Proyecto

Este proyecto corresponde a un **Portafolio Web Profesional Autoadministrable**, desarrollado como parte de la evaluación del módulo de Diseño y Desarrollo Web.

El sistema permite:

- Visualizar información personal
- Mostrar habilidades técnicas
- Gestionar proyectos
- Administrar contenido mediante un panel privado con autenticación

---

##  Tecnologías Utilizadas

###  Backend
- PHP 8+ (PDO)
- MySQL

###  Frontend
- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript (Vanilla + AJAX)

###  Servidor
- Apache (XAMPP / Hosting)
